Projects

Most recent work

Detection of Infected RBC cells and life stage of Malarial Parasites

This repository contains code for a malaria parasite detection system using Convolutional Neural Networks (CNNs). The system is designed to classify whether a given blood cell image is infected with malaria parasites (Plasmodium) or uninfected. The repository also privileges OpenCV and image processing techniques for detecting and extracting bounding boxes around objects in images

Google Colab Repository

Waiting Time Prediction in Queuing Systems using Machine Learning Algorithms

This project focuses on analyzing and predicting customer waiting times in a bank using statistical and machine learning methods. Key aspects of this work include data extraction, preprocessing, and feature engineering, followed by model training and evaluation. For predictive modeling, a multi-layer perceptron (MLP) neural network is used. Inputs are scaled to zero mean and unit variance to ensure robust training, leveraging methods for standardization.

Google Colab Repository

Banking Management System (C++)

This project is an extensive implementation of object-oriented programming (OOP) and data structures such as linked lists (for transaction history) and queues (for account closure requests) to efficiently handle dynamic data. Core OOP concepts, including encapsulation, inheritance, and abstraction, are employed to create a modular and maintainable design. The system supports essential banking functionalities like account creation, modification, balance inquiry, deposits, withdrawals, transaction history display, and account deletion.

GitHub Repository

Property Rental Management System (Oracle SQL)

The system provides robust functionalities for adding, updating, and deleting property records, utilizing PL/SQL stored procedures to handle complex database operations. Key features include user management, property details management, rent history tracking, and querying property status using unique identifiers such as property IDs and owner IDs. The database schema incorporates advanced normalization techniques with primary and foreign key constraints to ensure data integrity and consistency. This system is ideal for property managers.

SQL Query

VaR Modeling and Risk Analysis for Multi-Currency Portfolio

Applied financial concepts including Monte Carlo simulation to analyze portfolio risk. Statistical measures such as covariance, correlation, kurtosis, and skewness are employed to analyze portfolio returns, alongside mathematical constructs like Cholesky decomposition for matrix transformations. Quantified risk-return by calculating expected returns and correlation coefficients.
